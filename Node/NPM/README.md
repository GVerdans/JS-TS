# Sobre NPM

## Mover para o devDependencies
`npm install express --save-dev/prod`

## Instalar versão específica:
`npm install express@x.x.x`

## Ultima versão
`npm install express@latest`

## Atualizar versões
`npm update`

## Desinstalar
`npm unistall express`

## Listas as Dependencias
`npm ls`

## Verificar as att
`npm outdated`
